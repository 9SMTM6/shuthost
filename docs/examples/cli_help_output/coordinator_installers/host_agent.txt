Usage: scripts/coordinator_installers/host_agent.sh <remote_url> [--port PORT] [-- <install_args>]
Install ShutHost host agent from coordinator.

Arguments:
  remote_url     URL of the coordinator
  --port PORT    Port for WoL testing (default: 9090), also passed to install command
  -- <args>      Additional arguments for the host agent install command (except --port)
                 See repository path: docs/examples/cli_help_output/host_agent_install_linux.txt for subcommand help.
                 Note: init-system options may differ by platform, but the default is usually correct.
