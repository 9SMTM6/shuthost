Usage: scripts/enduser_installers/coordinator.sh [-t tag] [-b branch] [-h] [-- <binary-args>]
Install ShutHost coordinator binary.
Options:
  -t tag       Specify a release tag to download.
  -b branch    Specify a branch; tag will be 'nightly_release<branch>'.
  -h           Show this help message.
  -- <args>    Pass additional arguments to the coordinator install subcommand.
If no options, defaults to latest release.
