Usage: .\host_agent.ps1 [-Tag <tag>] [-Branch <branch>] [-Help] [-- <binary-args>]
Install ShutHost host agent binary.
Options:
  -Tag <tag>       Specify a release tag to download.
  -Branch <branch> Specify a branch; tag will be 'nightly_release<branch>'.
  -Help            Show this help message.
  -- <args>        Pass additional arguments to the agent install subcommand.
                   See repository path: docs/examples/cli_help_output/host_agent_install_linux.txt for subcommand help.
                   Note: init-system options may differ by platform, but the default is usually correct.
If no options, defaults to latest release.
