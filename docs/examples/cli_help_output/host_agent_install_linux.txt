Install the `host_agent` on the system

Usage: shuthost_host_agent install [OPTIONS]

Options:
  -p, --port <PORT>
          [default: 9090]

  -b, --broadcast-port <BROADCAST_PORT>
          [default: 5757]

  -c, --shutdown-command <SHUTDOWN_COMMAND>
          [default: "systemctl poweroff"]

  -s, --shared-secret <SHARED_SECRET>
          [default: <shared-secret>]

  -i, --init-system <INIT_SYSTEM>
          Possible values:
          - systemd:               Systemd init system (Linux)
          - openrc:                `OpenRC` init system (Linux)
          - self-extracting-shell: Generates a self-extracting shell script that embeds the compiled binary. The purpose is to keep the configuration readable (and editable) while being a single file that can be managed as one unit. You'll have to start the script yourself. [aliases: sh]
          - self-extracting-pwsh:  Generates a self-extracting `PowerShell` script that embeds the compiled binary. The purpose is to keep the configuration readable (and editable) while being a single file that can be managed as one unit. Note: Unlike the shell variant, the `PowerShell` script runs attached to the service process and does not automatically background itself. The installer will spawn the script in the background. [aliases: pwsh]
          
          [default: systemd]

  -n, --hostname <HOSTNAME>
          [default: <hostname>]

  -h, --help
          Print help (see a summary with '-h')
